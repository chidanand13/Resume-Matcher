 🧠 AI Resume Matcher

This project analyzes resumes and matches them with job descriptions using AI.

 Features
- Resume upload (PDF/TXT)
- Job matching score
- Missing skills detection
- Career suggestions

 Tech Stack
- Python
- Streamlit
- Scikit-learn

 Run
streamlit run app.py
